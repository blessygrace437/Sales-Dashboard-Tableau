# Sales Performance Dashboard using Tableau

## About This Project
This is my second data analytics project where I 
built an interactive Sales Dashboard using Tableau 
as part of my learning journey in Data Analytics.

## Dataset
- 500 sales records (self-created practice dataset)
- Fields: Region, Category, Product, Sales Rep, 
  Units Sold, Price, Total Sales, Profit

## What I Built
- Bar Chart → Sales by Region
- Pie Chart → Sales by Category
- Line Chart → Monthly Sales Trend (2024)
- Bar Chart → Profit by Sales Rep
- Combined all charts into one Dashboard

## Tools Used
- Tableau Desktop (Data Visualization)
- Microsoft Excel (Data Preparation)

## What I Learned
- How to import Excel data into Tableau
- Creating different chart types
- Building an interactive dashboard
- Analyzing sales data by region and category
